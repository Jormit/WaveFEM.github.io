About

WaveFEM is a program that began as hobby to understand the principles of electromagnetic simulation. The core simulation kernel is written from first principles enabling it to be highly performant and extendable. Without the overhead of a large corporation, we can offer simulation software at a fraction of the cost which is ideal for students and hobbyists.

Roadmap

Currently the core simulation code is done. This includes:

  • 3D Field Solver with integrated meshing
  • Adaptive mesh refinement
  • Radiation boundaries using PMLs
  • Farfield computation via integrated near to farifield transformation
  • Fast vector fitting implementation for interpolating sweeps
  • Support for importing multi-object .step files
  • Lumped ports and wave ports
  • Integrated 2D eigensolver supporting both TEM and TE modes
  • GUI which be used for material and port assignment
  • Parametric models defined via cadquery scripts
  • Importing planar formats such as .gds and gerber with inbuilt substrate editor

The coming features include:

  • Integrated Openfoam conjugate heat transfer solver
  • GPU accelerated solver via Nvidia CUDSS
  • Fast CNN-based simulation kernel for near-instant approximate planar simulations
  • Radar cross section analysis
