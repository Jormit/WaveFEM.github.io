About

WaveFEM began as a hobby to understand electromagnetic simulation. We wrote the core simulation kernel from scratch, making it highly performant and easy to extend. Without the high overhead of a large company, we can offer this powerful software at a fraction of the cost, making it ideal for students and hobbyists.

Roadmap

The core simulation code is finished and includes the following features:

  • 3D Field Solver with integrated meshing and adaptive mesh refinement
  • Radiation boundaries using Perfectly Matched Layers (PMLs)
  • Far-field computation via an integrated near-to-far-field transformation
  • Fast vector fitting for interpolating frequency sweeps
  • Support for importing multi-object .step files, as well as planar formats like .gds and .dxf, with a built-in substrate editor
  • Lumped and wave ports
  • Integrated 2D eigensolver that supports both TEM and TE modes
  • A GUI for material and port assignment
  • Parametric models defined using CadQuery scripts
