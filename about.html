About

WaveFEM began as a hobby to understand electromagnetic simulation. We wrote the core simulation kernel from scratch, making it highly performant and easy to extend. Without the high overhead of a large company, we can offer this powerful software at a fraction of the cost, making it ideal for students, researchers, and engineers.

Core Features

The core simulation code is finished and includes comprehensive electromagnetic simulation capabilities designed for accuracy and performance.

3D Field Solver

Integrated meshing and adaptive mesh refinement for optimal accuracy and computational efficiency across complex geometries.

Radiation Boundaries

Perfectly Matched Layers (PMLs) provide accurate absorption of outgoing waves for open-region electromagnetic problems.

Far-Field Computation

Integrated near-to-far-field transformation enables accurate antenna pattern and RCS calculations from near-field data.

Fast Vector Fitting

Advanced interpolation techniques for efficient frequency sweeps, reducing simulation time while maintaining accuracy.

Import & Design Tools

Seamless integration with industry-standard CAD formats and powerful parametric design capabilities for rapid prototyping and optimization.

Multi-Format Support

Import multi-object .step files, planar formats like .gds and .dxf, with a built-in substrate editor for complete design flexibility.

Port Configuration

Comprehensive lumped and wave port support with integrated 2D eigensolver for both TEM and TE mode analysis.

Material Assignment GUI

Intuitive graphical interface for material and port assignment, streamlining the simulation setup process.

Parametric Modeling

Create sophisticated parametric models using CadQuery scripts for design optimization and sensitivity analysis.
