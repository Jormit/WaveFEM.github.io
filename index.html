WaveFEM

An electromagnetic simulator for RF design

WaveFEM is an early stage finite element simulator for the simulation of high frequency components. It is able to calculate E and H fields, S-parameters, radiation patterns and visualise the results all within a simple unified interface.


Roadmap

The plan is to eventually recreate all the core functionality of commerical EM simulators, including:

  • Adaptive Mesh Refinement
  • Domain Decomposition Method
  • Radar Cross Section Computation
